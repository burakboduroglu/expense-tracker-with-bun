# Expense Tracker

This project is a simple expense tracker that uses new technologies to create a modern web application.

### Technologies have been used in this project:

- Bun
- React
- Drizzle
- Tailwind CSS
- Hono
- Kinde
- TypeScript
- RPC
- Tanstack
